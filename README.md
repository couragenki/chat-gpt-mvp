## このリポジトリについて
Chat GPTでいろんなアプリケーションの体験を試してみる

## 仮説

GPTのレコメンドの質はインプットの質で決まるが、大量の情報を毎回記載するのは大変。

そこでデータを保存したり、選択式にすることで手軽かつ高精度なレコメンドが返ってくるようにすることで、使用のハードルを避けてより日常で使えるように落とし込みたい。

## アプリ
Chat GPTを使ってどんな体験が作れそうかを以下のデモアプリで試してみる
1. チャット
1. 学習レコメンド(選択式)

## 技術スタック
- Next.js
- MUI
